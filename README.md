# Pufferfish Run 🐡

A mobile arcade game built with Phaser 3 and Capacitor.

## Concept
Play as a pufferfish navigating treacherous coral reefs. 
Tap to inflate and rise — but the bigger you get, the harder 
it is to squeeze through. The inflate mechanic creates a unique 
risk/reward loop not found in other arcade games.

## Tech Stack
- Phaser 3.90
- TypeScript
- Vite
- Capacitor (iOS/Android)

## Core mechanics
- Dynamic hitbox: inflating grows your collision radius
- Progressive difficulty: speed, gap size and spawn rate all scale independently
- Persistent high score via @capacitor/preferences
